# 🍉 3분수박 (3-Min Watermelon)

[English](README.md) | **한국어**

### ▶️ 지금 플레이: **https://crazythink.github.io/3-minute-watermelon**

귀엽고 청량한 **수박게임(Suika) 스타일 과일 합치기** 게임에 **로그라이크 + 발라트로식 점수 시스템**을 결합했습니다. 전부 **HTML 파일 하나**로 동작하며, 빌드 과정도 외부 의존성도 없습니다.

과일을 떨어뜨려 같은 과일을 합치고, 폭발적인 점수 배수를 쌓아 **3분** 제한 시간 안에 최고 점수에 도전하세요.

> 파일 하나만 열면 오프라인으로 플레이됩니다. 한국어 / English / 日本語 지원.

---

## ✨ 특징

- **수박게임식 물리 합치기** — 귀여운 둥근 과일 + 자체 고정 타임스텝 충돌 솔버(관통·무한 구르기 없음, 자연스러운 쌓임).
- **발라트로식 점수 인플레** — 점수 = `칩 × 배수`. 콤보·패시브·황금 터치·피버를 쌓아 점수를 폭증(보통 100만, 빌드를 잘 짜면 1억+).
- **로그라이크 카드** — 레벨업마다 5등급(일반 → 신화) 능력 카드 중 선택, 코인으로 **리롤**(강화로 무료 리롤 획득).
- **3가지 게임 모드**
  - 🍉 **클래식** — 작은 과일부터 합쳐 키우는 기본 모드. 강화·아이템 사용 가능.
  - 🔁 **리버스** — 수박부터 시작해 *아래로* 합칩니다. 과일이 **작을수록 고득점**.
  - 📅 **오늘의 도전** — 날짜 시드로 모두에게 같은 과일 순서. 강화·아이템 없이 공정하게 오늘의 최고 점수에 도전.
- **메타 성장** — 코인을 모아 영구 **강화**, 소비 **아이템**, 코스메틱 **배경 테마**, 반복 가능한 **환생(프레스티지, 영구 점수 배수)** 구매.
- **기록** — 누적 **통계**, 코인 보상이 있는 **도전과제**, 매일 바뀌는 **미션** + 연속 출석 보상.
- **카드 도감** — 모든 카드와 등급별 등장 확률 열람.
- **연출** — WebAudio로 합성한 BGM(잔잔한 메인 / 신나는 인게임) + 효과음, 모바일 햅틱, 신기록 시 폭죽, 점수 돌파 콜아웃, 탭 이탈 시 자동 일시정지, 결과 공유 카드.
- **다국어** — 🇰🇷 한국어 / 🇬🇧 English / 🇯🇵 日本語, 게임 내(🌐)에서 전환.
- **반응형** — 데스크톱과 모바일(세로) 레이아웃 모두 지원.

## ▶️ 플레이 방법

- 박스 안을 **클릭/터치**하면 현재 과일이 떨어집니다.
- **같은 과일** 두 개가 닿으면 다음 단계(클래식) 또는 이전 단계(리버스)로 **합쳐집니다**.
- **빨간 선 위로 과일이 쌓이지 않게** 하세요. 일정 시간 이상 넘기면 게임 오버.
- 합치면서 **레벨업**하면 능력 카드를 선택합니다.
- **3분** 안에 최대한 높은 점수를 내세요(아날로그 시계가 3분에 한 바퀴 차고, 넘으면 청록색으로 추가 표시).

## 🛠 기술

- **순수 JavaScript** + HTML5 **Canvas** — 프레임워크·번들러 없음.
- **WebAudio API** — 모든 음악·효과음을 런타임에 합성(오디오 에셋 없음).
- **localStorage** — 최고 점수, 코인, 강화, 아이템, 테마, 환생, 통계, 도전과제, 데일리 진행도, 언어·오디오 설정 저장.
- **Google Fonts**: `Jua`, `M PLUS Rounded 1c`, `Nunito` (유일한 외부 리소스).

HTML·CSS·JS·게임 데이터·번역까지 모두 `index.html` 한 파일에 들어 있습니다.
